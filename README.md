# Arendav Dispatcher

**Система управления заявками для логистической компании**

Сервис для диспетчеров и водителей с интеграцией Telegram бота для автоматизации работы с заявками.

## Основные возможности

### Для диспетчеров (Web-админка)
- Полный доступ к админке
- Создание и редактирование заявок
- Назначение водителей
- Просмотр всей статистики

### Для водителей (Telegram бот)
- Доступ только через Telegram бот
- Просмотр назначенных заявок
- Изменение статусов заявок
- Отправка путевых листов

## Быстрый старт

### Установка
1. Клонировать репозиторий
2. Установить зависимости: `composer install`
3. Настроить `.env` файл с базой данных
4. Запустить миграции: `php artisan migrate`
5. Настроить Telegram бота в `.env`:

### Настройка Telegram бота
1. Создать бота через @BotFather
2. Установить webhook: `php artisan telegram:webhook`
3. Зарегистрировать водителей в админке с указанием Telegram username

## Команды Telegram бота

- `/start` - Регистрация и начало работы
- `/mytrips` - Мои заявки и статистика  
- `/help` - Связь с диспетчером

## Поддержка

При возникновении проблем с ботом используйте команду `/help` для связи с диспетчером.
